# Website de Lanchonete - Golias Lanches

Projeto acadêmico desenvolvido com foco na aprendizagem de desenvolvimento web, utilizando HTML, CSS, Bootstrap e PHP.

O projeto foi criado com o objetivo de praticar a construção de interfaces web, navegação entre páginas e experiência do usuário, priorizando a parte visual e a interação da aplicação.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap
- PHP
- Git e GitHub

---

## Objetivo do Projeto

Desenvolver um website de lanchonete com múltiplas páginas e interface responsiva, simulando funcionalidades presentes em sistemas reais de pedidos online.

O back-end completo não foi implementado, pois o foco principal do projeto estava no desenvolvimento visual e na estruturação da aplicação web.

---

## Funcionalidades Desenvolvidas

* Navegação entre páginas  
* Visualização do cardápio  
* Estrutura visual de pedidos  
* Interface de carrinho de compras  
* Layout responsivo com Bootstrap  
* Organização de produtos em listas utilizando PHP e tabelas com HTML

---

## Funcionalidades Planejadas

- Fazer pedidos
- Adicionar produtos ao carrinho
- Finalizar compras
- Integração com banco de dados

---

## Estrutura do Site

O website possui 3 páginas principais:

### Início
Página inicial com apresentação da lanchonete e destaque visual dos produtos.

### Cardápio
Página destinada à exibição dos produtos disponíveis, organizada com tabelas e listas utilizando PHP.

### Sobre Nós
Página com informações sobre a proposta do projeto e da lanchonete.
